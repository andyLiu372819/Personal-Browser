# Milestones

## Milestone 1: Project Skeleton

- Create the repository structure.
- Add planning documents.
- Add default JSON data files.
- Add placeholder source modules.

## Milestone 2: First Browser Window

- Create the app entry point.
- Open a main window.
- Load the configured homepage.
- Show a web view.

## Milestone 3: Address Bar

- Add a combined URL and search input.
- Detect direct URLs.
- Convert domain-like inputs into HTTPS URLs.
- Send plain text queries to the default search engine.

## Milestone 4: Navigation

- Add back, forward, reload, and home controls.
- Keep the address bar synced with the current page.

## Milestone 5: Tabs

- Add a tab strip.
- Open new tabs.
- Close tabs.
- Switch between tabs.

## Milestone 6: Local Data

- Save history to `data/history.json`.
- Save bookmarks to `data/bookmarks.json`.
- Load settings from `data/settings.json`.

## Milestone 7: Personal Search

- Search local history and bookmarks.
- Show local results before falling back to a public search engine.
