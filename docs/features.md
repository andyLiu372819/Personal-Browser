# Features

## Core Features

- Load web pages.
- Search from the address bar using Nexus Search by default.
- Navigate backward and forward.
- Reload the current page.
- Return to a custom local homepage with a centralized search bar.
- Open and close tabs.
- Save bookmarks.
- Save browsing history.
- Edit settings.
- Open a proper settings page inside the browser.

## Nice-to-Have Features

- Private browsing window.
- Download manager.
- Reader mode.
- Basic ad blocking.
- Session restore.
- Custom new-tab page.
- Theme customization.

## Search Engine Features

Nexus Search searches crawled pages and the web:

- Search crawled page content.
- Search live web results through a provider adapter.
- Boost results from sites that appear in browsing history.
- Rank exact title matches first.
- Search from the custom homepage.
- Choose whether to display 10, 25, 50, 75, or 100 results.
- Keep the user on the Nexus Search results page even when there are no
  results, with direct provider links for DuckDuckGo and Google.
- Crawl selected websites with a configurable safe page/depth limit. The default
  crawl now indexes up to 75 pages, capped at 100 pages.

Later, it can grow into a stronger indexer with better snippets, recency scoring,
multiple web providers, and crawl scheduling.

## Theme Features

- Choose light or dark mode.
- Choose a built-in accent theme: Cyber Blue, Plasma Violet, Matrix Emerald, or
  Solar Amber.
- Choose a custom accent color.
- Apply the selected theme to the toolbar, homepage, settings page, history page,
  and search results page.
